# NIP-002 — Brand Identity

- Status: 📋 Open
- Area: Foundation / Brand
- Phase: 0
- Priority: High
- Date: 2026-05-11

## Context

NeuralMesh has no visual identity: no logo, color palette, typography, or brand guidelines.

## Proposal

Define and produce a complete visual identity:

### Sub-tasks

- [ ] NIP-002a: Finalize naming convention and definitive tagline
- [ ] NIP-002b: Design logo (variants: color, white, dark, favicon)
- [ ] NIP-002c: Define primary and secondary color palette (hex codes)
- [ ] NIP-002d: Select typography (heading + body fonts)
- [ ] NIP-002e: Produce brand guidelines document (`docs/brand/brand-guidelines.md`)
- [ ] NIP-002f: Export SVG/PNG assets for web and social media use

## Acceptance Criteria

- Logo available in SVG in at least 3 variants (color, mono, dark).
- Color palette defined with hex codes.
- Fonts selected and linked (Google Fonts or self-hosted).
- Brand guidelines documented.
- Favicon at 32x32 and 192x192 ready.

## Impact

- New folder: `assets/brand/`
- New: `docs/brand/brand-guidelines.md`

## Implementation Notes

- Motto: "Mine Intelligence, Not Waste."
- Naming: NeuralMesh — evokes neural networks + decentralized mesh networking.
- Tone: technical but accessible, not crypto-speculative.
- Aesthetic reference: dark tech, neon accents, minimal.
