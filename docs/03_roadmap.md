# Phase 1 — MVP

Goals:
- Desktop client
- Distributed task execution
- CPU/GPU mining
- Basic AI inference (LLM + OCR)
- Reward simulation
- OpenAI-compatible API endpoint

Technology:

- Go backend (libp2p, ONNX Runtime, Docker SDK)
- ONNX Runtime (CPU + CUDA)
- Docker sandboxing (no-network containers)
- PostgreSQL 16 + Redis 7

Budget Estimate:

- Out-of-pocket (hosting, tools): 2,000€–5,000€
- Developer labor (3–6 months senior Go engineer): 50,000€–150,000€
- **Realistic total: 55,000€–155,000€**
