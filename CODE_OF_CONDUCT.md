# Code of Conduct

## Our Pledge

We are committed to a welcoming, respectful environment for everyone regardless of experience, background, or identity.

## Expected Behavior

- Be respectful and constructive in discussions.
- Focus feedback on ideas and code, not people.
- Accept constructive criticism gracefully.
- Prioritize the health of the project and community.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Spam, self-promotion unrelated to the project.
- Publishing private information without consent.
- Misleading claims about NeuralMesh's capabilities.

## Enforcement

Violations may result in temporary or permanent ban from project spaces.
Report issues to the maintainers via GitHub Issues (private) or email.

## Scope

This Code of Conduct applies in all project spaces: GitHub, Discord, X/Twitter, and any official NeuralMesh channels.
